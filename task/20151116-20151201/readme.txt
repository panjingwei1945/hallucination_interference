1. 文件必须是MP3格式。

2. 目前版本只能播放5个文件。每次播放在5个文件中随机选择一个。播放中不可停止。
开始播放时播放按键变为报告按键。

3.命名规则：
	a. 文件名采用8.3的命名规则，文件名不能超过8个字符，拓展名不能超过3个字符。文件名与拓展名之间用"."分隔。

	b. 所有播放给被试的MP3文件名必须以数字开头，目前的版本只支持5个文件，所以命名必须以1-5开头，不可重复。

4. 下面是现在的刺激文件

	10k1_1k6.mp3
	1号文件，频率为100Hz + 200Hz + 400Hz + 800Hz + 1600Hz 每个刺激片段时长0.1s， 两个刺激间隔为0.4s， 刺激总时间为20s（重复40次），播放开始5.8s后开始播放刺激（前5.8s为说明和cue），25.5s播放完成，然后是约1.5s的播放完成的语音提示。


	20k5_8k.mp3
	2号文件，频率为500Hz + 1000Hz + 2000Hz + 4000Hz + 8000Hz, 其他条件同上。

	31k_16k.mp3
	3号文件，频率为1000Hz + 2000Hz + 4000Hz + 8000Hz + 16000Hz, 其他条件同上。

	42k.mp3
	4号文件，频率为纯音2000Hz, 其他条件同上。

	51k.mp3
	5号文件，频率为纯音1000Hz, 其他条件同上。




5. device 一共有5个led灯，充电时青色灯亮起，充电完成时变为绿色。充电时长约为3-4小时（因为电池自放电，所以有时充电完成后，一段时间充电灯又会变为青色，此时已进入下一个充电循环，所以只要冲3-4小时即可）。

按下播放按键,绿灯亮起。音量加按键红灯亮，音量减蓝灯亮。
